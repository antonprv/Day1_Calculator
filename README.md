# Calculator

Простое настольное приложение-калькулятор на Windows Forms.

## Описание

Приложение реализует базовый калькулятор с поддержкой основных арифметических операций.

Возможности:

* Сложение
* Вычитание
* Умножение
* Деление
* Работа с пользовательским вводом через UI

## Технологии

* C#
* Windows Forms
* .NET Framework 4.7.2

## Структура проекта

* `Logic/` — бизнес-логика калькулятора
* `Names/` — константы и перечисления (операции, символы, значения)
* `UI.cs` — пользовательский интерфейс
* `Program.cs` — точка входа в приложение
* `Properties/` — ресурсы и настройки проекта

## Запуск

1. Открыть решение `Calculator.slnx` в Visual Studio
2. Собрать проект (Build)
3. Запустить приложение (F5)

## Требования

* .NET Framework 4.7.2
* Visual Studio 2019 или новее

## Назначение

Учебный проект для демонстрации:

* разделения логики и интерфейса
* работы с Windows Forms
* реализации базовых арифметических операций
